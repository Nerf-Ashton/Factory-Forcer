## HOAs in Minecraft
Factory-Forcer forces players to place blocks with the tag "factory_forcer:indoor_only" underneath any collidable block as a roof.

If you're new to datapacks, check out this link: https://minecraft.wiki/w/Data_pack

Also, KubeJS is highly recommended as well to add tags to blocks for modpack makers.
